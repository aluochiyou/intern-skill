# Output Templates

Use these templates when the user benefits from a structured answer.

## Module Learning

```text
1. Current Learning Focus
2. Required Background
3. Code Structure And Execution Flow
4. Project Highlights
5. Engineering Details
6. Simple Baseline And Project Improvement
7. Must Know / Useful To Know / Skip For Now
8. Next Learning Steps
```

## Career Add-On

Use only when the user asks for job, interview, resume, internship, application, or portfolio framing.

```text
9. Interview Expression
10. Resume Candidate Points And Ownership Boundary
```

## Project Map

```text
Project Positioning:
Core User Actions:
Tech Stack:
Main Entrypoints:
Main Execution Paths:
Core Modules:
Supporting Modules:
Data Storage:
Test And Run Commands:
Learning Priority:
Next 1-3 Steps:
```

Prefer filling `Next 1-3 Steps` with a concrete recommendation. Avoid ending the first project map with a generic question such as "What do you want to learn next?"
